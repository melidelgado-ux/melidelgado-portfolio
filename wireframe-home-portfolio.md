# Wireframe Textual: Home Portfolio

## Objetivo de la Home

Presentar rapidamente quien soy, cual es mi forma de mirar el diseño y que tipo de trabajo puedo hacer. La Home debe combinar personalidad, claridad profesional y acceso rapido a los proyectos.

La lectura ideal:

1. Tiene una voz propia.
2. Es UX/UI con experiencia real.
3. Puede trabajar web, producto, visual, no-code e IA.
4. Tiene casos publicos fuertes.
5. Tiene mas recorrido disponible en archivo y entrevistas.

## 1. Header

### Contenido

Izquierda:

- Nombre / marca personal.

Centro o derecha:

- Work
- Learning
- Graphic Design
- About
- Contact

Acciones:

- Toggle ES / EN
- CTA pequeño: Contact

### Notas de layout

- Header simple y liviano.
- Puede ser sticky.
- Fondo claro.
- El logo/nombre puede tener un detalle manual pequeño.
- En mobile, menu colapsado.

## Notas Visuales Para Home

- Mantener mucho aire.
- Titulares grandes.
- Cuerpo de texto legible.
- Acentos manuales controlados.
- No usar muchas cards dentro de cards.
- Las miniaturas deben tener proporciones consistentes.
- El azul/indigo puede funcionar como tinta de anotaciones.
- El portfolio debe sentirse editorial, no plantilla SaaS ni CV plano.
